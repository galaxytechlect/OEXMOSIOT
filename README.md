# OEXMOSIOT
Otomatisasi exhaust fan dan monitoring suhu berbasis IOT
Rasberry PI 4

GPIO 13 = relay kipas 
GPIO 19 = relay lampu
GPIO 17 = sensor MQ2
GPIO 4 = sensor DHT11
